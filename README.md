# Обработка изображений с использованием OpenCV.js

## Описание

Веб-приложение предназначено для загрузки и обработки изображений прямо в браузере.  
Проект демонстрирует использование библиотеки компьютерного зрения **OpenCV.js** на стороне клиента.  
Пользователь может применять алгоритмы фильтрации и пороговой обработки, используя оптимизированные функции библиотеки, без установки дополнительного программного обеспечения.

## Возможности

- Загрузка изображений форматов **JPG**, **JPEG**, **PNG**, **BMP**, **WEBP**
- Отображение оригинального и обработанного изображений
- Просмотр изображений в увеличенном виде (модальное окно)
- Применение методов обработки на базе OpenCV:
  - Средний фильтр (сглаживание `cv.blur`)
  - Метод Оцу (автоматический глобальный порог)
  - Адаптивное среднее пороговое преобразование
  - Метод Бенсена (реализован через адаптивный Gaussian для обработки контраста)
- Полностью клиентская реализация (не требует серверной части)
- Адаптивный интерфейс

## Технологии

- **HTML5** — структура веб-страницы  
- **CSS3** — оформление и адаптивный дизайн  
- **JavaScript (ES6+)** — связующая логика интерфейса  
- **OpenCV.js** — библиотека компьютерного зрения (WebAssembly) для быстрой обработки  
- **Canvas API** — рендеринг изображений и обмен данными с OpenCV  

## Как это работает

1. При загрузке страницы инициализируется библиотека OpenCV.js.
2. Пользователь загружает изображение с помощью поля выбора файла.  
3. Изображение конвертируется в матрицу `cv.Mat` для обработки.  
4. При выборе метода вызывается соответствующая функция библиотеки:  
   - **Средний фильтр** — размытие изображения для устранения шумов.  
   - **Метод Оцу** — алгоритм находит оптимальный порог для разделения фона и объекта.  
   - **Адаптивное среднее** — порог рассчитывается для каждого пикселя на основе соседей.  
   - **Метод Бенсена** — адаптивная обработка, учитывающая локальный контраст изображения.  
5. Результат конвертируется обратно и выводится на второй canvas.  
6. При клике на изображение открывается модальное окно для детального просмотра.  

## Структура проекта

`index.html` — основной файл, содержащий разметку, стили и скрипт обработки с подключением OpenCV.

## Пример использования

1. Открыть файл `index.html` в браузере (требуется интернет для первой загрузки библиотеки).  
2. Загрузить изображение с компьютера.  
3. Выбрать метод обработки (например, «Метод Оцу»).  
4. Просмотреть результат на обработанном изображении.  

## Преимущества

- **Производительность:** Использование WebAssembly (OpenCV) быстрее обычного JS.  
- **Качество:** Применяются профессиональные алгоритмы компьютерного зрения.  
- **Безопасность:** Работает полностью в браузере, файлы не отправляются на сервер.  
- **Простота:** Не требует установки Python или C++.  
- **Совместимость:** Работает в современных браузерах (Chrome, Firefox, Edge).
